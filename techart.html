Tech Art Portfolio

Below are a collection of screenshots from projects where I was able to explore skills in game programming, shader development, particle systems, and animations. Click on a screenshot to see a video connected to my personal LinkedIn page highlighting each project.

Colour Theory Simulator

This is a project which allows a user to mix colours together as if they were physical paints. The user can store up to 3 mixed colours in 'blank' flasks, and see their mix on the canvas on the back wall as well as in the main flask. This project allowed me to tackle challenges such as particle collisions, colour mixing, and particle systems in general.

Paint the Town Rose

This is my project completed as part of my AIE studies. Paint the Town Rose is a game where the player must find each plain object in the level and touch it to re-paint it. This project allowed me to explore material swapping, particle systems, animations, and general game logic.
